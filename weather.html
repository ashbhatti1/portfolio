Weather Application

Back to Portfolio

The Weather Application provides real-time weather updates by using a third-party API (like OpenWeatherMap). It includes the following features:

  • Displays current weather conditions (temperature, humidity, wind speed).
  • Allows users to search for weather data by city name.
  • Responsive design for mobile and desktop.

This project was developed using HTML, CSS, JavaScript, and AJAX to fetch data dynamically.
